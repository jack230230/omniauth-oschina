# Omniauth::Oschina

oauth2 for www.oschina.net

## Installation

Add this line to your application's Gemfile:

    gem 'omniauth-oschina', git: 'https://github.com/jack230230/omniauth-oschina.git'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install omniauth-oschina
